1. 顺序访问P2P图书馆 API:

step1:
  • 列出第一页图书(每页10项)
  • 列出第二页图书(每页2项)
step2:
  • 登录为Calvin(注意保存Token)
  • Calvin请求借阅David的书 (手工替换URL中的Token为前一步保存的值)
  • 列出Cavlin借阅的图书(替换token)
step3:
  • 登录为David
  • David 确认图书借出
  • 列出David拥有的图书
step4:
  • David 确认图书归还，其他人可以借阅本书
  • 列出此书当前状态
other flow:
  • Calvin cancel the apply, replace the token
  • David reject the apply, replace the token
